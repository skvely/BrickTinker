About BrickTinker

After finishing my initial inventory as a new seller, I had a long list of things that bothered me about the BrickLink website. One day I realized that some of those problems could probably be fixed with a browser extension and that as a professional software developer, I could probably write it.

My experience is not in web development, but a few weeks later after lots of Googling, BrickTinker was born. The code is not beautiful but it gets the job done. If you're interested, it's publicly available on GitHub here.

 
BrickTinker's Future

This browser extension modifies the existing HTML, so any change on BrickLink's part could break any of the features. As long as I am using these features I will try to keep them up-to-date, but no guarantees. You can contact me through my BrickLink profile.

I am aware of BrickLink XP, but so far it doesn't seem to have any seller features. Hopefully when it is done this extension will no longer be needed.
